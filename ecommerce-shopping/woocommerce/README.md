## Getting Started

### Prerequisites

- Active Woocommerce account
- API credentials/access token
- Yshi account with integration permissions

### Installation

1. Navigate to Yshi's integration marketplace
2. Find and select Woocommerce
3. Click "Connect" or "Install"
4. Authenticate with your Woocommerce account
5. Grant necessary permissions
6. Configure your integration settings

### Configuration

```
// Example configuration
{
  "integration": "woocommerce",
  "auth": {
    "apiKey": "your-api-key",
    "apiSecret": "your-api-secret"
  },
  "settings": {
    // Add configuration options here
  }
}
```

### Issue: Connection Failed
**Solution**: Verify your API credentials are correct and active.

### Issue: Missing Permissions
**Solution**: Check your Woocommerce account permissions and API token scopes.

### Issue: Data Not Syncing
**Solution**: Verify webhook endpoints are configured correctly.
