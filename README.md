# Yshi Plugins

Yshi Plugins is a unified integration platform built on the [Yshi API](https://docs.yshi.co/), enabling developers to connect their tools and services directly to Yshi-powered bots, Mini Apps, and workflows.

## Overview

Yshi Plugins provides out-of-the-box integrations across five categories:

- **Developer Tools & DevOps** — GitHub, GitLab, PagerDuty, Sentry, Uptime Kuma
- **CRM, Sales & Productivity** — Salesforce, HubSpot, Jira, Notion, ClickUp, Trello, and more
- **E-commerce & Payments** — Shopify, WooCommerce, Stripe, PayPal, Magento
- **Content, Community & Support** — WordPress, Discord, Slack, Zendesk, Intercom, and more
- **Cloud Storage & Infrastructure** — Google Drive, Amazon S3, Dropbox, Nextcloud, OneDrive

## Yshi API Primitives

All integrations are built on four core Yshi API capabilities:

| Primitive | Purpose |
|-----------|---------|
| **Bot Webhooks** | Real-time event delivery and two-way messaging |
| **Mini Apps** | Embedded WebApp UIs inside Yshi chats |
| **OIDC OAuth** | Passwordless, single sign-on authentication |
| **Points System** | In-chat micro-transactions and reward mechanics |
